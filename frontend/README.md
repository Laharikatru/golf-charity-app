# Golf Charity App (React)

## 📌 Overview
The Golf Charity App is a web application built using React to support charity events in the golf community.

The application allows users to explore events, view details, and interact with a structured and user-friendly interface.

---

## 🚀 Features

- Browse charity golf events
- View event details
- Clean and responsive UI
- Structured navigation between pages
- Component-based architecture

---

## 🛠️ Tech Stack

- React.js
- JavaScript
- HTML
- CSS
